# CommunityEvents

A modern social events platform built with React and Vite, allowing users to create, manage, and join community events seamlessly.

## Live Site

[Visit CommunityEvents](https://social-development-event-59424.web.app/)

## Features

- **User Authentication**: Secure login and registration with email/password or Google OAuth
- **Event Creation**: Easily create and customize community events with detailed information
- **Event Management**: Full control over your created events - edit, update, or delete as needed
- **Event Discovery**: Browse and join upcoming events from the community
- **Responsive Design**: Optimized for all devices with a clean, modern interface
- **Dark/Light Theme**: Toggle between themes for personalized user experience
- **Real-time Updates**: Stay connected with the latest event information and notifications

## Tech Stack

- **Frontend**: React, Vite
- **Styling**: Tailwind CSS, DaisyUI
- **Authentication**: Firebase Auth
- **Backend**: Firebase (Firestore for data, Hosting for deployment)
- **Routing**: React Router

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up Firebase configuration in `.env.local`
4. Run the development server: `npm run dev`
